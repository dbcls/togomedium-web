# AGENTS.md

## このパッケージについて

- `@packages/api` は API の型・スキーマ定義を置くためのパッケージです。
- 現在は、このパッケージで zod から OpenAPI を書き出す現行構成を採用しています。
- 現状の `package.json` には専用 script がありません。作業時はルートの `pnpm run lint` `pnpm run fmt` `pnpm run test` を使って確認してください。

## 作業時の方針

- zod と OpenAPI まわりの型定義を壊さないよう、ランタイム実装よりも型の整合性を優先してください。
- まだ薄いパッケージなので、共通ロジックを安易に増やす前に `@packages/core` に寄せるべきかを確認してください。
- 新しい公開インターフェースを増やす時は、利用側がどのパッケージかを確認してから命名してください。
